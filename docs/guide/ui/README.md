Status: ACTIVE

## 1. Purpose

This subfolder groups UI-specific supporting guides so `docs/guide` stays clean and canonical flow docs remain easy to scan.

These docs support implementation and review but do not override canonical flow authorities in `docs/guide/README.md`.

## 2. Included Files

1. `UI_AUTOMATION_GUIDE.md`
2. `UI_CONFORMANCE_MATRIX.md`
3. `Clearance CSS Alignment Rules.md`
4. `Remit_Cleanup_Checklist.md`

## 3. Usage Order

1. Start with `UI_AUTOMATION_GUIDE.md` for contract and gates.
2. Use `UI_CONFORMANCE_MATRIX.md` for route-level conformance status.
3. Use `Clearance CSS Alignment Rules.md` and `Remit_Cleanup_Checklist.md` for focused implementation checks.

## 4. Execution Pairing

1. Operational run commands: `docs/automation/runbooks/`
2. Prompt templates: `docs/automation/templates/UI_AUTOMATION_PROMPT_TEMPLATE.md`
